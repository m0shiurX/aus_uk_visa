# start_frontend
AUS UK VISA applicaiton with html, scss and JavaScript (FRONTEND).

## User Manual:
   1. `git clone https://github.com/unimrgm/aus_uk_visa.git`
   2. `cd aus_uk_visa`
   3. `npm install`
   4. `npm start`
   
### Dependency : `git-bash and NodeJs`
#### Recomendation : Use Visual Studio Code as the code editor.
